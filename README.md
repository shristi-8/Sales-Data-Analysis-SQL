# SQL Sales Analysis

This project focuses on analyzing raw sales data using SQL.

## Project Workflow
- Imported raw sales data
- Cleaned missing and inconsistent values
- Formatted the dataset for analysis
- Wrote SQL queries to explore the data
- Performed sales analysis to generate business insights

## SQL Concepts Used
- SELECT
- WHERE
- GROUP BY
- ORDER BY
- Aggregate Functions (SUM, AVG, COUNT)
- JOINS
- CASE Statements
- Subqueries

## Project Outcome
The analysis provides insights into sales performance, customer behavior, product trends, and overall business performance using SQL.
